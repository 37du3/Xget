How to Use

Simply prefix your file URL with our service URL and the appropriate platform identifier:

Supported Platforms:

  • /gh/ - GitHub Raw Content
  • /gl/ - GitLab Files
  • /hf/ - HuggingFace Files
  • /kg/ - Kaggle Datasets
  • /url/ - Direct URLs

Examples

GitHub Example:

https://your-worker.workers.dev/gh/username/repo/branch/path/to/file

GitLab Example:

https://your-worker.workers.dev/gl/username/repo/-/blob/branch/path/to/file

Direct URL Example:

https://your-worker.workers.dev/url/https://example.com/file.zip

Limitations

  • Maximum path length: 2048 characters
  • Supported HTTP methods: GET, HEAD
  • Blocked file types: .php, .exe, .dll, .bat, .cmd, .sh
  • Cache duration: 24 hours
